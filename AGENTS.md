# First Chorus — AIアシスタント向け指示

**作業を始める前に、必ずリポジトリ直下の [HANDOFF.md](HANDOFF.md) を読んでください。**
プロジェクトの方針・アーキテクチャ・過去のバグから確立した設計原則・開発ワークフローがすべてまとまっています。

最低限の要点(詳細はHANDOFF.md):

1. **見本演奏なし**: AI生成の見本ソロ/フレーズの追加は禁止。再生してよいのは「譜面に表示された固定譜例」と「ユーザー自身が編集したフレーズ」だけ。
2. 内部の音程は常に**実音(Concert)MIDI**。移調は表示時のみ。五線譜・TAB・再生音の音程は常に一致させる。
3. 練習コースのデータは `src/data/courses.ts`(純データ)。STEPの説明文と譜面・再生内容を**完全に一致**させること。
4. 編集STEPの達成状態はstateにコピーせず、draftから**毎レンダー導出**する(`StepPractice.tsx`)。この構造を壊さない。
5. 変更後は `npx tsc --noEmit` と `npm run build` を通し、**実ブラウザで動作検証**(コンソールエラー0・モバイル390px)してからコミット。mainへのpushは本番へ自動デプロイされる。
6. 日本語・英語の両方を必ず更新する(`src/i18n.ts` とデータ内の`Bi`型)。
7. コミット・プッシュはオーナーの確認後に行うのが基本。
